1. Find Items

Context: Developing a C# application using SortedDictionary and List.

Functionalities

In the Program class, a public static SortedDictionary<String, long> itemDetails is provided. Implement the following:

Method Description
public SortedDictionary<String, long> FindItemDetails(long soldCount) Finds item details by sold count. Return the item as a SortedDictionary. If not found, return empty. Print "Invalid sold count" in Main if empty.
public List<String> FindMinandMaxSoldItems() Finds min and max sold items. Add minimum first, then maximum to a List and return it.
public Dictionary<string, long> SortByCount() Displays all items in ascending order by sold count. Return as a Dictionary.
